WikiQuiz is a web application that generates quiz questions automatically from Wikipedia articles.

## Features
- Enter a Wikipedia topic or URL
- Automatically generates fill-in-the-blank questions
- Flask backend + JavaScript frontend
- Uses NLTK for NLP processing

## Tech Stack
- Python 3.8
- Flask
- NLTK
- Wikipedia API
- HTML, CSS, JavaScript
